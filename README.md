# ربات کامل تلگرام سیگنال فیوچرز کریپتو

## شاخص‌های مورد استفاده:
- RSI
- EMA200
- MACD
- Bollinger Bands
- Ichimoku Cloud
- ATR برای Price Compression
- Fake Breakouts (شکست‌های جعلی)

## تایم‌فریم:
- تحلیل روند: 4H و 2H
- تشخیص ورود: 15m و 5m

## خروجی:
- سیگنال‌های با تأیید قوی: ⭐
- سیگنال‌های کم‌ریسک: 🚨

## نحوه اجرا:
1. ساخت فایل `.env`
2. نصب وابستگی‌ها:
```
pip install -r requirements.txt
```
3. اجرای ربات:
```
python main.py
```
یا deploy روی Railway.
